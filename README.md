# Amazon-sale-
Amazon Sales Dashboard 
**📌 Project Overview**

This project showcases an interactive Amazon Sales Dashboard built using Power BI to analyze and visualize sales data across multiple dimensions such as category, region, customer segment, and shipping mode.

**The goal of this dashboard is to provide clear business insights and support data-driven decision-making.**
